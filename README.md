# 🎓 小学四年级数学大通关

一个专为小学四年级学生设计的数学口算练习游戏，帮助孩子巩固口算能力，寓教于乐！

## 🎯 游戏特色

### 核心玩法
- **30道随机题目**：每次游戏从题库中随机抽取30道题
- **四选一选择题**：1个正确答案 + 3个干扰选项
- **实时统计**：显示当前题号、正确题数、正确率
- **进度条**：可视化显示答题进度

### 🎮 技能系统

游戏提供3种技能卡片，让答题更有策略性：

1. **🧑‍💼 求助爸爸**
   - 高亮显示正确答案（持续3秒）
   - 初始拥有1个

2. **❌ 排除错误**
   - 自动排除2个错误选项
   - 初始拥有1个

3. **🔄 换题**
   - 将当前题目替换为新题目
   - 初始拥有1个

### 🎁 奖励机制

- 每答对第6、12、18、24题后，随机获得1个技能卡片
- 第30题答对直接通关，无额外奖励

## 📚 题目类型

游戏包含四年级水平的多种题型（60+题库）：

### 基础运算
- ✅ 三位数加法（如：325 + 478 = ?）
- ✅ 三位数减法（如：856 - 479 = ?）
- ✅ 两位数乘法（如：23 × 45 = ?）
- ✅ 除法运算（如：144 ÷ 12 = ?）

### 进阶题型
- ✅ 混合运算（如：25 × 4 + 50 = ?）
- ✅ 带括号运算（如：(45 + 55) × 3 = ?）
- ✅ 简单应用题（如：小明有45元，买了3支笔...）
